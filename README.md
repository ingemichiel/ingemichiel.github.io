# Aartselaar — Woning Te Koop

Eenvoudige one-page landingspagina voor de particuliere verkoop van onze woning in Aartselaar.

## Structuur

```
aartselaar-te-koop/
├── index.html        # De volledige website (HTML + CSS + JS in één bestand)
├── photos/           # 30 foto's van de woning (01.jpg t/m 30.jpg)
└── README.md         # Dit bestand
```

## Lokaal bekijken

Dubbelklik op `index.html` — opent automatisch in je browser.
Of gebruik de Live Server-extensie in VS Code (rechtermuisklik op `index.html` → "Open with Live Server").

## Online zetten via GitHub Pages (gratis)

1. Push deze map naar een GitHub-repo (bv. `aartselaar-huis`).
2. Open de repo op github.com → **Settings** → **Pages**.
3. Bij "Source" → kies branch `main`, folder `/ (root)` → **Save**.
4. Na 1-2 minuten staat de pagina online op:
   `https://JOUW_GEBRUIKERSNAAM.github.io/aartselaar-huis/`

## Wat moet je nog aanpassen in `index.html`?

Open `index.html` en zoek-en-vervang (Ctrl+H in VS Code):

| Zoek | Vervang door |
|---|---|
| `+32475123456` | Je echte telefoonnummer (zonder spaties of streepjes) |
| `+32 475 12 34 56` | Hetzelfde nummer mét spaties (voor de weergave) |
| `jouwadres@email.be` | Je echte e-mailadres |

Deze placeholders komen meerdere keren voor — vervang ze allemaal.
